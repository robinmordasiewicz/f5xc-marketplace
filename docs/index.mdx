## Key Features

- **Automation first** — Automate repetitive console tasks with intelligent
  browser automation powered by Claude's understanding of intent.
- **AI-powered intelligence** — Translate natural language commands to precise
  console actions. Describe what you want, and Claude handles the details.
- **Enterprise security** — Deploy in enterprise environments with Azure AD SSO,
  multi-tenant support, and secure authentication flows.
- **Open source** — Contribute workflows, report issues, and shape the future of
  F5 automation under the MIT license.

## Quick Start

Get up and running in 60 seconds:

### Step 1: Add marketplace

```bash
/plugin marketplace add robinmordasiewicz/f5xc-marketplace
```

### Step 2: Install plugin

```bash
/plugin install f5xc-console
```

### Step 3: Start automating

```text
You: "Navigate to the shared namespace and show me all HTTP load balancers"
Claude: Navigating to shared namespace... Found 3 HTTP load balancers...
```

## What You Can Do

- **Load Balancers** — Create, configure, and manage HTTP/HTTPS load balancers with natural language commands.
- **Origin Pools** — Set up origin pools, configure health checks, and manage backend servers.
- **WAF Policies** — Deploy and configure Web Application Firewall policies for application security.
- **Multi-Tenant Navigation** — Navigate complex multi-tenant environments with intelligent namespace switching.
